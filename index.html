Ataxx

Eric Khumalo
CS61B

Goal

Ataxx is a two-person game played with red and blue pieces on a 7-by-7 board. Designed and wrote the entire game for 2 players. I also used the Min-Max Algorithm to write the AI which is able to find a forced win that is within five moves of a given position. I employed Alpha Beta Pruning to improve execution time. AI had a 83.4 % win rate against 1200 fellow student AIs.
